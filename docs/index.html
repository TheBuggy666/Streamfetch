🚀 StreamFetch v2.0

Created by: Buggy_666

StreamFetch is a free and open-source desktop YouTube downloader supporting playlists, multi-links, and multiple formats. Built-in auto-updater, version info, and donation support included.

✨ Features

  • 🎵 MP3, M4A, WAV, MP4, WEBM downloads
  • 📥 Multi-link & Playlist support
  • 🔄 Auto-updates yt-dlp at launch
  • 🖥️ GUI interface with dark theme
  • 📄 Link to README & version display

📦 How to Use

Just download the ZIP, extract, and run install.bat or build the EXE via build.bat.
